# KKS to KK Card Online

在线 KKS 卡转 KK 卡，支持打包为 ZIP 和单独下载

![](./screenshot/screenshot.png)

## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Compile and Minify for Production

```sh
pnpm build
```
